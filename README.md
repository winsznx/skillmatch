# SkillMatch

Decentralized freelancer marketplace connecting clients and talent on Base and Stacks blockchains.

## Features

- Post freelance jobs with budgets
- Accept jobs as freelancer
- Mark jobs as completed
- Skill registry system
- Payment escrow

## Tech Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Base**: Solidity 0.8.20, Foundry, Reown wallet
- **Stacks**: Clarity v4, Clarinet, @stacks/connect

## Getting Started

```bash
pnpm install
pnpm dev
```
